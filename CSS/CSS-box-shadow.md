# CSS3 阴影

## box-shadow

box-shadow: offSetX(X 偏移) offSetY(Y 偏移) blur(模糊度) spread(半径) color position

- offSetX，向下为正，向上为负
- offSetY，向下为正，向上为负
- blur，越大越模糊，0 为全部显示
- spread，越大越宽，负数向内

box-decoration-break，当文本内容发生截断时的样式

- 默认 slice，css 样式效果也发生截断
- clone,css 样式会重新渲染，效果将保持相同
